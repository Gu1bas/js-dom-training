 # JS DOM Training 🚀

Coleção de projetos para prática de manipulação de DOM e lógica de programação. Feito em novembro de 2025

## 🛠️ Projetos Inclusos

### 1. Mini Carrinho de Compras
Um sistema que simula a adição de produtos a um carrinho, com atualização de preço em tempo real.
* **Destaques Técnicos:** Uso de `querySelectorAll`, criação dinâmica de elementos com `classList` e template strings, e função de atualização de total (`updateTotal`).
* **Funcionalidades:** Adicionar produtos, remover itens específicos e cálculo automático do valor total.

### 2. Calculador de Despesas
Ferramenta para gestão de gastos simples com divisão entre pessoas.
* **Destaques Técnicos:** Manipulação de eventos `click` e `change`, cálculos matemáticos e formatação de moeda com `toFixed(2)`.

## 💻 Tecnologias
* HTML5
* JavaScript (ES6+)
* CSS3
